# Implementation of Unix-Shell in C

### Features:
- Custom UI
- Colourful Aids
- Tab completetion
- Deals with accidental spaces and tabs
	
### Commands Implemented:
- File system related:
	- mkdir  
	- cat  
	- ls
	- cd
	- rm 
	- rmdir
	- pwd
	- cp
	- mv
	- touch

- File Operations:
	- wc
	- sort (-o, -r)
	- uniq 

- System Information:
	- ifconfig
	- lscpu
	- free
	- w
	- ps (-p, -pid)

- Shell UI related:
	- exit
	- clear
	- about

- Executable:
	- gcc


## To Run:
```
Compile shell.c and modular_shell.c files
run ./shell or run make
```
